Mouse

Class: -

Note: this page is under development. The information below was written for Xtreme3D v2 and may not correspond to the Xtreme3D v3.

Simple functions of the mouse.

MouseGetX

Real MouseGetX =();
Returns the X coordinate of the mouse cursor on the upper left corner of the screen.
The function is similar display_mouse_get_x.

MouseGetY

Real MouseGetY =();
Returns the y coordinate of the mouse cursor on the upper left corner of the screen.
The function is similar display_mouse_get_y.

MouseSetPosition

Real MouseSetPosition =( x,y as real );
Specifies the coordinates of the mouse cursor on the upper left corner of the screen.
The function is similar display_mouse_set.
X,Y coordinates.

MouseShowCursor

Real MouseShowCursor =( mode as real );
Enables or disables the display of the mouse cursor.
Mode - true or false.
