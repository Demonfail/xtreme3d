Color

Class: -

Functions for Working with color.

MakeColorRGB

Real MakeColorRGB =( r,g,b as real );
Creates a color from the values of the channels R, G, B and returns it. Feeds are defined by the integer numbers from 0 to 255.
R, G, B - integer color channels.

MakeColorRGBFloat

Real MakeColorRGBFloat =( r,g,b as real );
Creates a color from the values of the channels R, G, B and returns it. Feeds are defined by the material numbers from 0.0 to 1.0.
R, G, B - float the color channels.
