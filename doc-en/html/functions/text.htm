Text

Class: -

Functions for working with strings and texts.

TextRead

String = TextRead( filename as string );
Loads the text file in the string and returns it.
Filename is the name of the file.

TextConvertANSIToUTF8

String = TextConvertANSIToUTF8( text as string );
Converts a string from the ANSI encoding to UTF-8 (Unicode). The input string is not modified, feature creates in the memory of a new string and returns it. In order to avoid memory leaks it is not recommended to cause this feature constantly - convert a String once in the beginning of the program and use it.
Text - the text.
