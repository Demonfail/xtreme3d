Blur

Class: TGLBlur

The blur (Blur), all the familiar effect. In this case, we are talking about motion blur - �������� when driving, which visually occurs when a fast rotating or moving point of view. The introduction of this effect may give realism in some game situations. Motion blur is often used, for example, in the races.
The principle of the effect of the blur is the storage and rendering several previous frames with the gradual scaling and reduction of transparency (and, in some cases, and color). It is, of course, could not affect the performance. The blur, therefore, should not be used too often.

Creates a new object of blur and returns its id.
Target is the id of the object, which must be blurred. Will be blurred and all descendants of this object.
Parent - the id of the parent.

BlurSetPreset

Real BlurSetPreset =( blur,p as real );
Specifies the style of blurring (p). You can choose which one is more suitable for a particular situation.
The available values of p:
PNone = 0 - lack of blur
PGlossy = 1 - "Shine" - moderate blur and brightness, without tinting
PBeastView = 2 - "the eyes of the beast" - the blood-red tint, weak blur
POceanDepth = 3 - "In the depths of the ocean" - the thick blue tint, a strong blur
PDream = 4 - "dream" is a strong slow motion blur, weak brightness, without tinting
POverBlur = 5 - "�������������" - a strong blur, a strong brightness, without tinting.
Blur - the object id blur
P - style of blur.

BlurSetOptions

Real BlurSetOptions =( blur,delta,left,top,right,bottom as real );
Specifies the rendering options of blur. Delta - the delay time before imposing the next blur frame. The below this value, the better the effect, but below the performance. Left, top, right, bottom - offset borders frames left, up, right, down. Changing the offset, can be horizontal, vertical, or diagonal blur.
For the delay and offset are usually very small values are within the 0.001 - 0.015. The Offset allows for negative values.
Blur - the object id blur
Delta - Delay in seconds
Left, top, right and bottom is offset to the left, top, right, down.

BlurSetBlendingMode

Real BlurSetBlendingMode =( blur,bm as real );
Sets the sets the mixing of blurring (BM). You have the following options: bm
BmOpaque = 0 - lack of transparency and blending (full replacement)
BmTransparency = 1 - use transparency (alpha blending)
BmAdditive = 2 - used summation of flowers
BmAlphaTest50 = 3 - use the alpha test at 50% (Transparency below 0.5 is treated as full transparency above - Full Opacity)
BmAlphaTest100 = 4 - Use of alpha-test at 100% (Transparency below 1.0 is treated as full transparency above - Full Opacity)
BmModulate = 5 - used modulation (multiplication) colors.
Blur - the object id blur
Bm - mode mixing.
