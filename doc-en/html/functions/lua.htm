Lua

Class: TLua

Starting with version 3.6, Xtreme3D includes support for Lua - the popular script programming language. For the Lua scripts are available with all of the functions of the Xtreme3D. You can use this opportunity to realize the dynamic logic. For example, the behavior of the game objects can be scripts that allows you to play, to create a fashion. In fact, virtually all of the game can be entirely written in Lua.
To use the Lua interpreter to put lua.dll in the directory with the project.
You can learn more about the language of the online https://www.lua.org.

LuaManagerCallFunction

Real = LuaManagerCallFunctionlua( as real, name as string );
Performs the function of Lua and returns 1, if this function was defined in the previously executed the script. Otherwise returns 0. Only supported functions without arguments.
Lua is the id of the Lua Manager
Name - the name of the function.
