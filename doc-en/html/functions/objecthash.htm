ObjectHash

Class: TObjectHash

ObjectHash object is intended for storage of any other objects Xtreme3D in the hash table - the special data structure provides the opportunity to access objects by name. This can be useful, for example, to simplify the game logic, if you upload a scene from a file (i.e., automatically create a lot of objects, unknown to the program in advance).

ObjectHashCreate

Real ObjectHashCreate =( );
Creates a new hash table and returns its id.

ObjectHashSetItem

Real ObjectHashSetItem =( hash as real, key as string, obj as real );
Saves the object in the hash table under the specified key (name).
Hash - id hash tables
Key is a string key
Obj is the object ID.

ObjectHashGetItem

Real ObjectHashGetItem =( hash as real, key as string );
Returns an object that is stored in a hash table under the specified key. If the table there is no such key, the function returns 0.
Hash - id hash tables
Key is a string key.

ObjectHashDeleteItem

Real ObjectHashDeleteItem =( hash as real, key as string );
Removes from the table object, saved under the specified key. If there is no such key, then nothing happens.
Hash - id hash tables
Key is a string key.

ObjectHashGetItemCount

Real ObjectHashGetItemCount =( hash as real );
Returns the number of stored in a hash table objects.
Hash - id hash tables.

ObjectHashClear

Real ObjectHashClear =( hash as real );
Clears the contents of hash tables. The table itself remains, and it can be used.
Hash - id hash tables.

ObjectHashDestroy

Real ObjectHashDestroy =( hash as real );
Destroys the hash table. Use it after calling this function.
Hash - id hash tables.
