TrailCreate

Real TrailCreate =( object,parent as real );
Creates a new trail and returns its id.
Object is the id of the target, which must be applied loop effect
Parent - the id of the parent.

TrailSetObject

Real TrailSetObject =( trail,object as real );
Change the target object, which must be applied loop effect
Trail - the object id trail
Object-ID new target object.

TrailSetAlpha

Real TrailSetAlpha =( trail,alpha,fade as real );
Specifies the transparency of the plume.
Trail - the object id trail
Alpha - the transparency of the
Fade - Determines whether you need to make the loop slowly disappearing: true or false (1 and 0, respectively).

TrailSetLimits

Real TrailSetLimits =( trail,vl,tl as real );
Defines the limitations of the plume.
Trail - the object id trail
The vl - maximum cable length
Tl - time before the disappearance of the plume (in seconds).

TrailSetMinDistance

Real TrailSetMinDistance =( trail,distance as real );
Sets the distance at which passes the target should create a regular segment of the plume. The higher this value, the better the effect.
Trail - the object id trail
Distance is the distance.

TrailSetUVScale

Real TrailSetUVScale =( trail,uv as real );
There are no details.
Trail - the object id trail
Uv - there are no details.

TrailSetMarkStyle

Real TrailSetMarkStyle =( trail,ms as real );
Specifies the style of the plume (ms). You have the following options: ms
MsUp = 0 - Flex is always aligned with the vertical axis
MsDirection = 1 - loop is similar to the radar screen: to move in a circle on a local object center trail, following the target
MsFaceCamera = 2 - Flex is always aligned with the camera, and it therefore seems surround (the default value).
Trail - the object id trail
Ms - the style of the plume.

TrailSetMarkWidth

Real TrailSetMarkWidth =( trail,width as real );
Specifies the size of the plume.
Trail - the object id trail
Width is the size (default: 1).

TrailSetEnabled

Real TrailSetEnabled =( trail,mode as real );
Enables or disables the ribbon cable.
Trail - the object id trail
Mode - true or false (1 and 0, respectively).

TrailClearMarks

Real TrailClearMarks =( trail as real );
