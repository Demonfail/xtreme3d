Shadowvolume

Class: TGLShadowVolume

Shadowvolume object allows you to build a real volumetric shadows in real time! In contrast to the shadowplane, shadows here is not limited to the plane, they may fall on any objects. This method of shading is an object that defines the volume of the inside which points are in the shade. Unfortunately, he is extremely resource-intensive and can lead to tremendous impact in performance. Therefore, it is recommended to apply the shadowvolume on a small number of objects.

ShadowvolumeCreate

Real ShadowvolumeCreate =( parent as real );
Creates a new object of shadowvolume and returns its id. The objects to be drop shadow, you need to make the descendants of shadowvolume.
Parent - the id of the parent.

ShadowvolumeSetActive

Real ShadowvolumeSetActive =( sv,mode as real );
Enables or disables the shade. In the off state shadowvolume consume computing resources, therefore, should be turned off when the shadows do not fall within the field of view.
Sv - the object id shadowvolume
Mode - true or false (1 and 0, respectively).

ShadowvolumeAddLight

Real ShadowvolumeAddLight =( sv,light as real );
Adds a light source, from which the drop shadow. You can add multiple sources, but this will affect performance.
Sv - the object id shadowvolume
Light is the id of the light source.

ShadowvolumeRemoveLight

Real ShadowvolumeRemoveLight =( sv,light as real );
Removes the source of light from the shadowvolume.
Sv - the object id shadowvolume
Light is the id of the light source.

ShadowvolumeAddOccluder

Real ShadowvolumeAddOccluder =( sv,object as real );
Adds the object, which should drop shadow. The object may simultaneously and discard, and to take the shade.
Sv - the object id shadowvolume
The object is the object ID.

ShadowvolumeRemoveOccluder

Real ShadowvolumeRemoveOccluder =( sv,object as real );
Removes the object from the shadowvolume.
Sv - the object id shadowvolume
The object is the object ID.

ShadowvolumeSetDarkeningColor

Real ShadowvolumeSetDarkeningColor =( sv,color,alpha as real );
Specifies the color and transparency of shadows.
Sv - the object id shadowvolume
Color,alpha - the color and transparency.

ShadowvolumeSetMode

Real ShadowvolumeSetMode =( sv,svm as real );
Sets the shadows (SVM):
SvmAccurate = 0 is the default mode
SvmDarkening = 1 - in this mode the working color and transparency of Shadows
SvmOff = 2 - Shadow of the disabled.
Sv - the object id shadowvolume
The Svm - the shadows.

ShadowvolumeSetOptions

Real ShadowvolumeSetOptions =( sv,showvolumes,cachesilhouettes,scissorclips,worldscissorclip as real );
Specifies the shadowvolume options.
Sv - the object id shadowvolume
Showvolumes - display of shadow volumes: true or false (1 and 0, respectively). The default value is 0
Cachesilhouettes - Enables or disables the caching of silhouettes: true or false (1 and 0, respectively). Caching can improve performance, when the objects are motionless on the shadows. The default value is 1.
Scissorclips - Enables or disables the trim the shadows within the screen area, the area of influence of the source of light: true or false (1 and 0, respectively). This crop increases performance, if the light source has a limited area of influence, and too far removed from the Shadows should not be rendered. The default value is 1.
Worldscissorclip - Enables or disables the trim the shadows within the screen area, corresponding to the object that has the shadow: true or false (1 and 0, respectively). The default value is 0.
