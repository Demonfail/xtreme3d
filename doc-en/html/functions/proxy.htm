Proxy

Class: TGLProxyObject, TGLActorProxy, TGLMultiProxy

A proxy is a special type of object, which does not have its own graphics for drawing - he draws the graph of another object, but uses for this own transformation. In other words, the proxy - this is the best way to create multiple copies of the 3D model, without duplicating it in memory.

ProxyObjectCreate

Real ProxyObjectCreate =( object,parent as real );
Creates a new proxy with graphics of the specified object.
The object is the object id
Parent - the id of the parent.

ProxyObjectSetOptions

Real ProxyObjectSetOptions =( proxy,copyeffects,copyobjects,copytransform as real );
Specifies the proxy option.
Proxy - proxy id
Copyeffects - Specifies whether to copy the effects of the specified object (we are talking about particle systems attached to the object)
Copyobjects - Specifies whether to copy the descendants of the object
Copytransform - Specifies whether to copy the local transformation of the object.
