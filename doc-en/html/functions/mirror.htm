Mirror

Class: TGLMirror

Mirror (mirror) - the plane, showing the other objects. Reflections is achieved by drawing the mirror copies of objects within a rectangular or ������������� clipping plane.

MirrorCreate

Real MirrorCreate =( target,parent as real );
Creates a new mirror and returns its id. The Mirror size by default, 1x1.
Target is the id of the target, which should be reflected in the mirror (all his descendants will also be reflected).
Parent - the id of the parent.

MirrorSetObject

Real MirrorSetObject =( mirror,target as real );
Changes the object, which should be reflected in the mirror.
Mirror Mirror id
Target is the id of the object.

MirrorSetOptions

Real MirrorSetOptions =( mirror,stencil,opaque,planeclipping,clearzbuffer as real );
Specifies the options of the mirror.
Mirror Mirror id
Stencil - true or false (1 or 0). If the option is enabled, the reflection will be truncated on the plane. Default: 1
Opaque - true or false (1 or 0). If the option is disabled, the mirror plane will be transparent. Default: 0
Planeclipping - true or false (1 or 0). If this option is enabled, the objects in the mirror, will not give back-reflection. Default: 0
Clearzbuffer - true or false (1 or 0). If the option is enabled, the reflection will be completely overlap the objects in the mirror. Default: 0.
Realistic mirror will give a combination of 1,0,1,1.

MirrorSetShape

Real MirrorSetShape =( mirror,ms as real );
Specifies the shape of the mirror - this may be a rectangular plane (msRect) or disk (msDisk).
Mirror Mirror id
Ms is a form of mirrors: msRect = 0 or msDisk = 1.

MirrorSetDiskOptions

Real MirrorSetDiskOptions =( mirror radius,,slices as real );
If the mirror is in the form of disc, this function sets the radius and the number of segments of the disc.
Mirror Mirror id
Radius is the radius of the disk
Slices - the number of segments of the disc.
