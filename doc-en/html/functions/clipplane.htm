ClipPlane

Class: TGLClipPlane

The object ClipPlane (clipping plane allows to render objects only on one side of the target plane. This can be useful when implementing some of the effects - for example, the reflections on the water in the on-screen space. ClipPlaneCreate

Real ClipPlaneCreate =( parent as real );
Creates a new clipping plane and returns its id.
Parent - the id of the parent.

ClipPlaneEnable

Real ClipPlaneEnable =( cp,mode as real );
Activates or deactivates the clipping plane. If the plane is deactivated, the amputation is not performed.
Cp - id clipping plane
Mode - true or false (1 and 0, respectively).

ClipPlaneSetPlane

Real ClipPlaneSetPlane =( cp,px, py, pz,nx,NY,nz as real );
Specifies the parameters of the amputation plane.
Cp - id clipping plane
Px, py, pz - the coordinates of the center of the plane
Nx,NY,nz - the normal plane. Amputation is subject to all of the graphics, the underlying on the back side of the plane normal.
