Input

Class: -

The functions that allow you to read the user input from the keyboard and mouse.

KeyIsPressed

Real KeyIsPressed =( vk as real );
Returns 1, if the key is depressed, and 0 otherwise.
Vk is the key code. The codes match the usual constants vk_left, vk_enter and so forth. If you need to check that you have pressed an alphanumeric key, to obtain the use ord - for example, ord('a').

MouseGetX

Real MouseGetX =( );
Returns the X coordinate of the mouse cursor on the upper left corner of the screen (no window). The function is similar display_mouse_get_x.

MouseGetY

Real MouseGetY =( );
Returns the y coordinate of the mouse cursor on the upper left corner of the screen (no window). The function is similar display_mouse_get_x.

MouseSetPosition

Real MouseSetPosition =( x,y as real );
Specifies the coordinates of the mouse cursor on the upper left corner of the screen (no window). The function is similar display_mouse_set.
X,Y coordinates.

MouseShowCursor

Real MouseShowCursor =( mode as real );
Enables or disables the display of the mouse cursor. Please note that the call of this function is only useful when the display cursor to the Global Game Settings - If the cursor is disabled in the settings, this function does not display it.
Mode - true or false.
