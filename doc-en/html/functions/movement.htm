Movement

Class: TGLMovement, TGLMovementPath, TGLPathNode

The Object Movement (movement) allows you to specify the objects of the trajectory of the movement. The trajectory are called ways (path). The path may represent a set of line segments, or spline (the wave) - cubic spline, Bezier and NURBS. MovementCreate

Real MovementCreate =( object as real );
Creates an object Movement, which will control the movement of the specified object and returns its id.
The object is so id of the object.

MovementAddPath

Real MovementAddPath =( movement as real );
Adds the object movement on a new path and returns the id (not the index!).
Movement is the id of object Movement.

MovementSetActivePath

Real MovementSetActivePath =( movement,index as real );
Specifies the current (active) path of the object Movement. MovementStart function always starts with the first node in this path.
Movement is the id of object Movement
Index - the index of ways, starting with 0. The default value is 0.

MovementStart

Real MovementStart =( movement as real );
Starts the movement so the object. The position and rotation of the object is always reset to the initial node of the current path.
Movement is the id of object Movement.

MovementStop

Real MovementStop =( movement as real );
Stops the movement. After stopping, controlled by an object, you can move the usual functions.
Movement is the id of object Movement.

MovementAutoStartNextPath

Real MovementAutoStartNextPath =( movement,mode as real );
Determines whether to switch automatically to the next path when the current one.
Movement is the id of object Movement
Mode - true or false (1 and 0, respectively). By default, the option is enabled.

MovementPathSetSplineMode

Real MovementPathSetSplineMode =( path,lsm as real );
Specifies the type of spline path (lsm). Learn more about the types of splines you can read in the section Lines.
LsmLines = 0 is a set of line segments
LsmCubicSpline = 1 - cubic spline (the default setting)
LsmBezierSpline = 2 - Bezier Spline
LsmNURBSCurve = 3 - NURBS.
Path is the path id
Lsm is a type of spline.
