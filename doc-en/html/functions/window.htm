Window

Class: TForm

Functions for creating additional windows and management. You can render in the created the window graphics Xtreme3D, if making when creating the type of the corresponding identifier (HWND) using the WindowGetHandle. Please note that before the end of games all created by the window you want to delete function of the WindowDestroy, otherwise an error will occur.

WindowCreate

Real WindowCreate =( x,y,width,height,resizeable as real );
Creates a window and returns its id.
X,Y position of the window on the screen
Width,height - the width and height of the window
Resizeable - whether the user can change the size of the window (true or false). Please note that changing the window size does not change the size of the connected view, to do it manually.

WindowCenter

Real WindowCenter =( window as real );
Places the window in the center of the screen.
Window - id window.

WindowResize

Real WindowResize =( window,x,y,width,height as real );
Specifies the position and size of the window. Please note that changing the window size does not change the size of the connected view, to do it manually.
Window - The window id
X,Y position of the window on the screen
Width,height - the width and height of the window.

WindowGetPosition

Real WindowGetPosition =( window,index as real );
Returns the position of the window (upper left corner).
Window - The window id
Index - if this option is 0, the function returns the X coordinate of the if 1 - Y.

WindowGetSize

Real WindowGetSize =( window,index as real );
Returns the width or height of the window.
Window - The window id
Index - if this option is 0, the function returns the width, if 1 is the height.

WindowGetHandle

Real WindowGetHandle =( window as real );
Returns the ID of the window (HWND).
Window - id window.

WindowSetTitle

Real WindowSetTitle =( window as real, title as string );
Specifies the window title.
Window - The window id
Title - the title.

WindowDestroy

Real WindowDestroy =( window as real );
Removes the window.
Window - id window.
