LinesCreate

Real LinesCreate =( parent as real );
Creates a new line (a chain of nodes) and returns its id.
Parent - the id of the parent.

LinesAddNode

Real LinesAddNode =( line,x,y,z as real );
Adds a new node in the line and assigns a sequence number. Counting starts from scratch.
Line - the line id
X,y,z coordinates the local host.

LinesDeleteNode

Real LinesDeleteNode =( line,ind as real );
Removes the assembly to a preset number.
Line - the line id
Ind - the number of the node.

LinesSetColors

Real LinesSetColors =( line,linecolor,linealpha,nodecolor,nodealpha as real );
Specifies the color of the lines and nodes.
Line - the line id
Linecolor,linealpha - color and transparency of lines
Nodecolor,nodealpha - color and transparency of the nodes.

LinesSetSize

Real LinesSetSize =( line,linewidth,nodesize as real );
Specifies the width of the lines and the scale of the display of nodes.
Line - the line id
Linewidth - the width of the Lines
Nodesize - scale nodes.

LinesSetSplineMode

Real LinesSetSplineMode =( line,lsm as real );
Specifies the type of lines (lsm):
LsmLines = 0 - Direct
LsmCubicSpline = 1 - cubic spline
LsmBezierSpline = 2 - Bezier Spline
LsmNURBSCurve = 3 - NURBS
LsmSegments = 4 - dotted line.
Line - the line id
Lsm - type lines.
