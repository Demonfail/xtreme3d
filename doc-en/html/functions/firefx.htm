FireFX

Class: TGLFireFXManager, TGLBFireFX

FireFX - the system of particles, modeling fire. While using the FireFX, you can recreate and other effects: smoke, sparks, explosions, rain, fireworks, and Bengal lights, etc.
FireFX system consists of a finite number of particles - small objects-sprites, depicting the tiny colored dots. Usually the particles moving in the same direction corresponding to the Direction vector object (the truth, you can "blow up" the system - then set free would throw the particles in all parties).

In some cases, it is necessary to create multiple identical effects. Not very easy to repeat all the settings for each of them. Specifically for such a situation, a manager of The FireFX. Manager - it is kind of a server on which performed all the calculations related to the effect. A separate system of FireFX is the customers using settings specified for the manager. You simply create a manager, configure it as you want, and then add any number of systems in all the right places you in the scene. The changes made to the manager settings will automatically affect all their reporting systems.

Outcolor,outalpha - color and transparency of the periphery of the particles.

FireFXExplosion

Real FireFXExplosion =( manager,isp,maxsp,lbf as real );
"Blasts" system of particles. The set free would throw the particles in all parties. If you call this function once, there will be a single explosion, after which the system will come back to normal. But if the cause of its constantly, explosion and will last, recalling the Bengali fire.
Manager - id FireFX manager
Isp - initial velocity scattering
Maxsp - Maximum Speed Scattering
Lbf - the radius of the Sphere, beyond which the explosion should not exit.
