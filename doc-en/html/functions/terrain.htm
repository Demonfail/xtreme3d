The Terrain

Class: TGLBitmapHDS, TGLTerrainRenderer

Xtreme3D supports the drawing of landscape (terrain), consisting of a grid of rectangles - ������. When rendering the landscape takes into account the distance from the camera to reduce the number of polygons at remote sites (LOD). This process is called tessellation.
To create a landscape requires map heights (HDS - Height Data Source) - black and white, light areas which means the increase in the height of the surface, dark - decline. The Hds is loaded from the image file. The color depth of the textures should correspond to the 8 bits per pixel (256 Color, Grayscale). Otherwise, the landscape will be staggered.
