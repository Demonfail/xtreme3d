ThorFX

Class: TGLThorFXManager, TGLBThorFX

ThorFX is a specialized system of particles, oriented to the work with a solid line. Using The ThorFX can create incredibly realistic lightning, electric shocks, the rays of the laser, plasma flows and other effects that use the direct or broken line.
ThorFX system consists of two components: the rod, or core (core), and the glow (glow). Rod - a solid line, filled in one color, and the glow is a collection of individual particles (sprites), evenly distributed along the entire length of the rod. When a sufficient number of particles they visually merge in solid volume, and would seem to be that the rod is lit, as the neon tube.
For the re-establishment of the effect of the electric shock (it is called the effect of Tesla, on behalf of the outstanding physics of Nikola Tesla) is activated by the vibration and offset - and before us is not neon tube, and this lightning!
The position of the system in space is set its own coordinates, as well as coordinates of the target point, which should "beat" lightning.

In some cases, it is necessary to create multiple identical effects. Not very easy to repeat all the settings for each of them. Specifically for such a situation, a manager of The ThorFX. Manager - it is kind of a server on which performed all the calculations related to the effect. A separate system of ThorFX is the customers using settings specified for the manager. You simply create a manager, configure it as you want, and then add any number of systems in all the right places you in the scene. The changes made to the manager settings will automatically affect all their reporting systems.

ThorFXManagerCreate

Real ThorFXManagerCreate =();
Creates a new manager of ThorFX and returns its id.

ThorFXCreate

Real ThorFXCreate =( manager,object as real );
Creates a new system of ThorFX and binds it to the specified object.
Manager - id ThorFX manager
Target is the id of the object.

ThorFXSetColor

Real ThorFXSetColor =( manager,incolor,inalpha,outcolor,outalpha,ccolor,calpha as real );
Specifies the color of the system.
Manager - id ThorFX manager
Incolor,inalpha - color and transparency of the center of illumination
Outcolor,outalpha - color and transparency of the periphery of the illumination
Ccolor,calpha - color and transparency of the rod.

ThorFXEnableCore

Real ThorFXEnableCore =( manager,mode as real );
Enables or disables the rod.
Manager - id ThorFX manager
Mode - true or false (1 and 0, respectively).

ThorFXEnableGlow

Real ThorFXEnableGlow =( manager,mode as real );
Enables or disables the glow.
Manager - id ThorFX manager
Mode - true or false (1 and 0, respectively).

ThorFXSetMaxParticles

Real ThorFXSetMaxParticles =( manager,particles as real );
Specifies the number of particles in the shining. The higher the number, the more realistic looks glow.
Manager - id ThorFX manager
Particles - the number of particles (from 0 to 255).

ThorFXSetGlowSize

Real ThorFXSetGlowSize =( manager,size as real );
Specifies the size of the illumination.
Manager - id ThorFX manager
Size is the size.

ThorFXSetVibrate

Real ThorFXSetVibrate =( manager,vibrate as real );
Specifies the level of vibration of the rod (fluctuations). Zero means the absence of vibration.
Manager - id ThorFX manager
Vibrate - the level of vibration.

ThorFXSetWildness

Real ThorFXSetWildness =( manager,wildness as real );
Specifies the offset of the rod on the center line. An electrical shock to "jerks", reinforcing the impression of the effect. This often can be observed both in nature and in scientific experiments.
Zero means the absence of bias.
Manager - id ThorFX manager
Vibrate - the level of vibration.

ThorFXSetTarget

Real ThorFXSetTarget =( manager,x,y,z as real );
Specifies the target point system (that is, the point at which the beats of lightning). The coordinates of the point is always defined locally, that is, on its own the coordinate axes of the system. If the coordinates asked absolutely, all created system of a manager were sent to the same point that agree, usually quite undesirable.
Manager - id ThorFX manager
X,y,z coordinates of the point.
