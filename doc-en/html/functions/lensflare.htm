Lensflare

Class: TGLLensFlare

Lensflare - this effect, which often can be seen in the camera lens when shooting bright light sources. Ppearance of light is repeated many times within the system of lenses in the form of a rosy circles and rings, sometimes the hexagons. In computer gaming is often used to convey a sense of space and atmosphere. The most frequently lensflare you can see in games with open space: this is The ��������������, race, some of the FPS and so on.
In the Xtreme3D has an excellent built-in lensflare, which you can set up on their own taste and get the maximum realistic effect. It consists of the main flash, similar to the star, and several circles. The main flash unit corresponds to the light source and includes several elements that can ������������ in any combination: large beams, small beams, ring (it is called the halo), a soft glow.
Creating a lensflare object, you must place it in the same location as the source of light.

LensflareSetResolution

Real LensflareSetResolution =( lf,res as real );
Specifies the resolution - the number of segments that comprise the elements of lensflare. The higher the value, the "�������" will be all circles and rings. And reducing the resolution, you can instead of circles to get the aforementioned hexagons.
Lf is the object id lensflare

LensflareSetElements

Real LensflareSetElements =( lf,glow,ring,streaks,rays,secs as real );
Specifies the combination of elements lensflare. Disabling Some of the elements that you can, for example, to make simply beautiful star in the sky, the light from car headlights, etc.
Lf is the object id lensflare
Glow - Enables or disables the radiance: true or false (1 and 0 respectively)
The ring is enables or disables the ring: true or false (1 and 0 respectively)
Streaks - Enables or disables the big rays: true or false (1 and 0 respectively)
Rays - Enables or disables the small rays: true or false (1 and 0 respectively)
Secs - Enables or disables the circles: true or false (1 and 0, respectively).

LensflareSetGradients

Real LensflareSetGradients =( lf,ind,color1,alpha1,color2,alpha2 as real );
Specifies the color and transparency elements. You must specify the color and the transparency of the center and edges for radial gradients, which is filled with the element.
You must also specify the index of the element, which should change color:
1 = Radiance
2 = Ring
3 = Large rays
4 = small rays
5 = circles.
Lf is the object id lensflare
Ind - the index of an element
Color1,alpha1 is the color and the transparency of the Center
Color2,alpha2 is the color and the transparency of the edges.
