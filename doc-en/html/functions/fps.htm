FPS

Class: TGLFPSMovementManager

FPS is a special, fully automated method of detection and processing of the clashes, optimized for gaming in the first person. Although there is nothing preventing use it in games from third person. The FPS environment is extremely simple to manage, so it can be safely recommended as a base system clashes for both novice and experienced users Xtreme3D.

FpsManagerCreate

Real FpsManagerCreate =();
Creates the manager of the FPS and returns its id.

FpsManagerSetNavigator

Real FpsManagerSetNavigator =( manager,nav as a real );
The navigator indicates that you want to use in the environment of FPS. If this is not done, moving objects will be impossible.
Manager - manager id FPS
Nav is the id of the navigator.

FpsManagerSetMovementScale

Real FpsManagerSetMovementScale =( manager,scale as real );
Specifies the factor speed. In fact, this option only affects the force of gravity.
Manager - manager id FPS
Scale - the coefficient of speed.

FpsManagerAddMap

Real FpsManagerAddMap =( manager,object as real );
Adds the card. Map is an object of free form (freeform), representing the geometry of the level used for collision detection. This geometry may not be the same as the one that is directly visible to the player in the form of architectural buildings and other objects as well as from the point of view of optimization is much more profitable to provide complex elements of the level as a simple invisible "boxes".
Manager - manager id FPS
The object is the object ID.

FpsManagerRemoveMap

Real FpsManagerRemoveMap =( manager,object as real );
Removes the selected card. In fact, card remains untouched, but it ceases to take into account the collision handler FPS.
Manager - manager id FPS
The object is the object ID.

FpsManagerMapSetCollisionGroup

The object is the object id

FpsSetManager

Real FpsSetManager =( object,manager as real );
Specifies the manager of the FPS for the object. After this procedure, the object will be taken into account when checking for collisions, and his movement is controlled by the functions of the FPS.
The object is the object id
Manager - manager id FPS.
