Grid

Class: TGLXYZGrid

Grid (grid) is a set of intersecting line segments, forming a uniform rectangular grid, two-dimensional or three-dimensional. This facility is convenient for use in debugging and subsidiary purposes, when it is necessary to visualize the coordinate plane.
