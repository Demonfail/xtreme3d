DirectX
A set of Apis for game programming in the environment of MS Windows. Includes 2d rendering (DirectDraw and Direct3D) graphics, user interaction through input devices is a keyboard, mouse, joystick (DirectInput), as well as the sound (DirectSound API).
The first version of DirectX was released in 1994.
The main competitor to Direct3D is a OpenGL.

See also the API, OpenGL.
