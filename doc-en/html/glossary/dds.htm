The DDS - raster graphics format developed by Microsoft for use with the DirectX API. It is widely applied in multimedia applications, including 3D games.
In contrast to conventional compression formats such as JPEG and PNG, DDS textures stored in memory in a compressed form, and the decompression is done in hardware in real time. Thus, the use of the DDS allows requirements to video memory, Reduce boot time textures, as well as to increase the speed of rendering by reducing the number of accesses to memory.
