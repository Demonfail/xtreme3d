The API

Application Programming Interface | An Application Programming Interface
The interface between the program and the equipment, or between the program and other programs. The main gain from the use of API is the portability (it is possible to write one and the same software for different platforms), the stability (the program's ability to be compatible with future versions of hardware and software), and simplicity.
