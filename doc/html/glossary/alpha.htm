Alpha | Альфа
Коэффициент прозрачности.
В описание цвета (RGB) может входить специальный компонент, называемый альфа-каналом, который отвечает за прозрачность данного цвета. В линейном цветовом пространстве значение альфы 0 означает полную прозрачность, 1 - полную непрозрачность.

См. также Color, Color Model.
