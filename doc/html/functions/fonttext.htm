Font & Text

Класс: TGLWindowsBitmapFont, TGLHUDText

ВНИМАНИЕ: страница находится в разработке. Информация ниже была написана для Xtreme3D v2 и может не соответствовать Xtreme3D v3.

Шрифт и текст. Xtreme3D поддерживает три типа текстовых объектов: плоский, объемный и экранный. Экранный текст отображается поверх отрендеренной картинки и применяется для представления некоей текстовой информации в игре: количество жизней или патронов, различных сообщений и пр. Плоский и объемный текст расположены внутри трехмерного пространства - объемный, в отличие от плоского, имеет толщину.
Все типы текста используют специальные шрифтовые объекты, хранящие изображения текстовых символов - букв, цифр и знаков препинания. Эти изображения можно задать двумя способами: сгенерировать из системных векторных шрифтов Windows или загрузить из файла. Второе более предпочтительно, так как Вы можете нарисовать в графическом редакторе шрифт любого цвета и сложности, с любыми символами и на любом языке, в то время как поддержка системных шрифтов сильно ограничена (поддерживается только латиница). Но у векторных шрифтов есть одно неоспоримое преимущество - масштабируемость.
Обратите внимание, что Xtreme3D не распознает в строковых значениях символ # как переход на следующую строку. Вместо этого рекомендуется использовать тринадцатый символ ASCII - chr(13). Например:
TextSetText(text, "line 1" + chr(13) + "line 2").

BmpfontCreate

real = BmpfontCreate( width,height,hspace,vspace,intx,inty,start,end as real );
Создает новый шрифт и возвращает его id.
width,height - ширина и высота символа в пикселях
hspace,vspace - промежуток между символами по горизонтали и вертикали в пикселях
inx,inty - координаты начала отсчета символов (крайняя верхняя левая точка)
start,end - номера первого и последнего буквенного символа. Для латиницы эти значения соответствуют 32 и 95.

BmpfontLoad

real = BmpfontLoad( font as real,material as string );
Загружает изображение шрифта из текстуры указанного Материала.
font - id шрифта
material - имя Материала.

WindowsBitmapfontCreate

real = WindowsBitmapfontCreate( fontname as string, size as real );
Загружает векторный шрифт Windows и возвращает его id.
fontname - название шрифта, например: 'Arial'
size - размер шрифта.

FlatTextCreate

real = FlatTextCreate( font as real, string as string , parent as real );
Создает плоский трехмерный текст и возвращает его id.
font - id шрифта
string - текстовая строка для отображения
parent - id родителя для текстового объекта (0 - отсутствие родителя).

HUDTextCreate

real = HUDTextCreate( font as real, string as string , parent as real );
Создает экранный текст и возвращает его id.
font - id шрифта
string - текстовая строка для отображения
parent - id родителя для текстового объекта (0 - отсутствие родителя).

HUDTextSetRotation

real = HUDTextSetRotation( text,angle as real );
Задает угол поворота экранного текста.
text - id текстового объекта
angle - угол поворота в градусах.

SpaceTextCreate

real = SpaceTextCreate( font as real, string as string , extrusion,parent as real );
Создает объемный трехмерный текст и возвращает его id.
font - id шрифта
string - текстовая строка для отображения
extrusion - толщина текста
parent - id родителя для текстового объекта (0 - отсутствие родителя).

SpaceTextSetExtrusion

real = SpaceTextSetExtrusion( text,extrusion as real );
Задает толщину объемного текста.
text - id текстового объекта
extrusion - толщина текста.

TextSetColor

real = TextSetColor( text,color,alpha as real );
Задает цвет и прозрачность для экранного или плоского текста.
text - id текстового объекта
color - цвет
alpha - прозрачность.

TextSetFont

real = TextSetFont( text,font as real );
Задает шрифт для экранного или плоского текста.
text - id текстового объекта
font - id шрифта.

TextSetText

real = TextSetText( text as real , string as string );
Задает текстовую строку для отображения текстовым объектом любого типа.
text - id текстового объекта
string - текстовая строка.
