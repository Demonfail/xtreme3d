Mesh

Класс: TGLFreeForm

ВНИМАНИЕ: страница находится в разработке. Информация ниже была написана для Xtreme3D v2 и может не соответствовать Xtreme3D v3.

Функции меша (mesh) относятся, как правило, к объектам свободной формы и работают непосредственно с их геометрией.

MeshCountObjects

real = MeshCountObjects( object as real );
Возвращает количество мешей в модели.
object - id объекта.

MeshRotate

real = MeshRotate( object,x,y,z,angle as real );
Поворачивает модель вокруг заданной оси. Обратите внимание, что этот поворот относится только к геометрии и никак не влияет на обычный поворот объекта по осям XYZ.
object - id объекта
x,y,z - единичный вектор оси поворота
angle - угол поворота.

MeshExplode

real = MeshExplode( object,sp,max,x,y,z as real );
"Взрывает" модель: полигоны, составляющие его, разлетаются в стороны. "Собрать" меш обратно нельзя.
object - id объекта
sp - постоянная скорость разлета
max - максимальная скорость разлета
x,y,z - единичный вектор движения. Если эти значения соответствуют нулю, полигоны разлетятся в направлении своих нормалей, т.е. во все стороны.
