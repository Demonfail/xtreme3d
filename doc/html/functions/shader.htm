Шейдер в Xtreme3D - это спецэффект для материала. Есть пять разных шейдеров: Bump, Cel, HiddenLine, MultiMaterial, Outline, TexCombine, Phong и GLSL. Для применения шейдера к материалу используется функция MaterialSetShader. При помощи шейдеров на языке GLSL вы можете программировать свои собственные спецэффекты.

ShaderEnable

real = ShaderEnable( shader,mode as real );
Включает или выключает шейдер.
shader - id шейдера
mode - true или false (1 и 0 соответственно).

BumpShaderCreate

real = BumpShaderCreate( );
Создает шейдер рельефа (bump mapping) и возвращает его id. При применении шейдера первая текстура материала должна быть картой нормалей.
Данный шейдер требует поддержки расширений GL_ARB_vertex_program и GL_ARB_fragment_program.

BumpShaderSetMethod

real = BumpShaderSetMethod( shader,bm as real );
Задает метод реализации шейдера рельефа: bmDot3TexCombiner (при помощи расширения ARB_texture_env_dot3) или bmBasicARBFP (при помощи вершинной и фрагментной программ).
shader - id шейдера
bm - метод: bmDot3TexCombiner = 0 или bmBasicARBFP = 1.

BumpShaderSetSpecularMode

real = BumpShaderSetSpecularMode( shader,sm as real );
Задает модель бликовой компоненты рельефа: smOff (без бликов) или smBlinn (модель Блинна), smPhong (модель Фонга). В режиме bmDot3TexCombiner блики не работают.
shader - id шейдера
sm - модель бликовой компоненты: smOff = 0, smBlinn = 1 илиsmPhong = 2.

real = BumpShaderSetSpace( shader,bs as real );
Сведения отсутствуют.
shader - id шейдера
bs - сведения отсутствуют.

BumpShaderSetParallaxOffset

real = BumpShaderSetParallaxOffset( shader,offset as real );
Сведения отсутствуют.

offset - сведения отсутствуют.

BumpShaderSetOptions

real = BumpShaderSetOptions( shader,texture2,texcoords2,lightatten,parallax as real );
Задает опции шейдера рельефа.
shader - id шейдера
texture2 - указывает, использовать ли вторую текстуру материала для диффузной компоненты - true или false (1 и 0 соответственно)
texcoords2 - указывает, использовать ли второй набор текстурных координат модели для диффузной текстуры - true или false (1 и 0 соответственно)]
lightatten - указывает, использовать затухание источников света при расчете освещенности - true или false (1 и 0 соответственно)
parallax - указывает, использовать ли параллактическое отображение (parallax mapping) - true или false (1 и 0 соответственно).

CelShaderCreate

real = CelShaderCreate();
Создает так называемый cel-шейдер и возвращает его id. Этот тип шейдера используется для создания специального эффекта мультипликационного изображения. Он включает отрисовку контуров и упрощение модели освещения (свет и тень представлены четко разграниченными пятнами).
Данный шейдер требует аппаратной поддержки расширения GL_ARB_vertex_program.

CelShaderSetLineColor

real = CelShaderSetLineColor( shader,color as real );
Задает цвет контура в cel-шейдере.
shader - id шейдера
color - цвет.

CelShaderSetLineWidth

real = CelShaderSetLineWidth( shader,width as real );
Задает толщину контура в cel-шейдере.
shader - id шейдера
width - ширина.

HiddenLineShaderCreate

real = HiddenLineShaderCreate();
Создает шейдер скрытых линий. Скрытые линии - это ребра полигонов, из которых состоят объекты. Обычно они не отображаются, но этот шейдер позволяет их увидеть.

HiddenLineShaderSetLineSmooth

real = HiddenLineShaderSetLineSmooth( shader,mode as real );
Включает или выключает анти-алиасинг линий (по умолчанию отключен).
shader - id шейдера
mode - true или false (1 и 0 соответственно).

HiddenLineShaderSetSolid

real = HiddenLineShaderSetSolid( shader,mode as real );
Включает или выключает отображение самого материала (по умолчанию отключено).
shader - id шейдера
mode - true или false (1 и 0 соответственно).

HiddenLineShaderSetSurfaceLit

real = HiddenLineShaderSetSurfaceLit( shader,mode as real );
Включает или выключает влияние освещения на материал с шейдером.
shader - id шейдера
mode - true или false (1 и 0 соответственно).

HiddenLineShaderSetFrontLine

real = HiddenLineShaderSetFrontLine( shader,width,color,p,f as real );
Задает параметры передних линий. Передние линии принадлежат полигонам, нормали которых повернуты в сторону наблюдателя.
shader - id шейдера
width - толщина
color - толщина
p - cведения отсутствуют (по умолчанию 65535)
f - cведения отсутствуют.

HiddenLineShaderSetBackLine

real = HiddenLineShaderSetBackLine( shader,width,color,p,f as real );
Задает параметры задних линий. Задние линии принадлежат полигонам, нормали которых повернуты в противоположную сторону от наблюдателя.
shader - id шейдера
width - толщина
color - толщина
p - cведения отсутствуют (по умолчанию 65535)
f - cведения отсутствуют.

MultiMaterialShaderCreate

real = MultiMaterialShaderCreate( matlib as real );
Создает мультиматериальный шейдер и возвращает его id. Он позволяет создать материал, состоящий из нескольких слоев других материалов, с собственными текстурами и параметрами. Эти материалы загружаются из отдельной библиотеки.
Этот чрезвычайно мощный шейдер будет главным образом полезен для имитации поверхностей, состоящих из нескольких веществ с различными свойствами: например, деревянный ящик, окованный металлом, или камень с вкраплениями лавы или кристаллов. Но ему можно найти и другие применения.
matlib - id библиотеки материалов.

OutlineShaderCreate

real = OutlineShaderCreate( smooth as real );
Создает контурный шейдер и возвращает его id. Используется для простейшего мультипликационного эффекта - отрисовки контуров вокруг объекта.
smooth - cведения отсутствуют.

color - цвет.
