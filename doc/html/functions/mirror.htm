Mirror

Класс: TGLMirror

ВНИМАНИЕ: страница находится в разработке. Информация ниже была написана для Xtreme3D v2 и может не соответствовать Xtreme3D v3.

Зеркало (mirror) - плоскость, отражающая другие объекты. Эффект отражения достигается путем отрисовки зеркальных копий объектов внутри прямоугольной плоскости отсечения.

MirrorCreate

real = MirrorCreate( target,parent as real );
Создает новое зеркало и возвращает его id. Размер зеркала по умолчанию 1х1.
target - id целевого объекта, который должен отражаться в зеркале (все его потомки также будут отражаться).
parent - id родителя.

MirrorSetObject

real = MirrorSetObject( mirror,target as real );
Меняет объект, который должен отражаться в зеркале.
mirror - id зеркала
target - id объекта.

MirrorSetOptions

real = MirrorSetOptions( mirror,stencil,opaque,planeclipping,clearzbuffer as real );
Задает опции зеркала.
mirror - id зеркала
stencil - true или false (1 или 0). Если опция включена, отражение будет обрезано по плоскости. По умолчанию: 1
opaque - true или false (1 или 0). Если опция выключена, плоскость зеркала будет прозрачной. По умолчанию: 0
planeclipping - true или false (1 или 0). Если опция включена, объекты, находящиеся за зеркалом, не будут давать обратного отражения. По умолчанию: 0
clearzbuffer - true или false (1 или 0). Если опция включена, отражение будет полностью перекрывать объекты, находящиеся за зеркалом. По умолчанию: 0.
Реалистичное зеркало даст комбинация 1,0,1,1.
