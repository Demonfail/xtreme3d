Freeform

Класс: TGLFreeform

ВНИМАНИЕ: страница находится в разработке. Информация ниже была написана для Xtreme3D v2 и может не соответствовать Xtreme3D v3.

Объект свободной формы (Freeform) - неанимированная модель, загружаемая из файла 3DS (3D Studio), OBJ (Maya), LWO (Lightwave), GLSM (GLScene Mesh), BSP (Quake), MS3D (Milkshape), CSM (Cartography Shop), LMTS (LMTools) или DMF (DeleD).
Объекты такого типа используются обычно в качестве игровых уровней, декораций и интерактивных предметов.

FreeformCreate

real = FreeformCreate( filename as string, parent as real );
Создает новый объект свободной формы и возвращает его id.
filename - путь к файлу модели
parent - id родителя для объекта свободной формы (0 - отсутствие родителя).

FreeformPointInMesh

boolean = FreeformPointInMesh( freeform,x,y,z as real );
Возвращает true (1), если заданная точка находится внутри меша объекта свободной формы. Функция хорошо подходит, например, для проверки столкновений объекта с пулей.
freeform - id объекта свободной формы
x,y,z - координаты точки.

FreeformSphereSweepIntersect

real = FreeformSphereSweepIntersect( freeform,sphere,radius,vel as real );
Возвращает true (1), если сфера заданного радиуса вокруг заданного объекта, движущегося с заданной скоростью, при следующем шаге столкнется с поверхностью объекта свободной формы.
freeform - id объекта свободной формы
sphere - id объекта
radius - радиус сферы (относительно поверхности объекта)
vel - скорость движения объекта.

FreeformToFreeforms

real = FreeformToFreeforms( freeform,start,count,parent as real );
Разбивает объект свободной формы на его составляющие - меши, каждый из которых становится отдельным объектом свободной формы.
freeform - id объекта свободной формы
start - индекс начального меша, с которого следует начинать разбивку
count - количество отделяемых мешей
parent - id родителя для полученных объектов.

FreeformSetSecondTextureCoords

real = FreeformSetSecondTextureCoords( freeform1,freeform2 as real );
Применяет текстурные координаты объекта freeform2 в качестве вторых текстурных координат объекта freeform1. Второй набор текстурных координат обычно используется для наложения так называемой карты освещения (lightmap).
freeform1 - id объекта свободной формы
freeform2 - id объекта свободной формы.
