Спрайт (Sprite) - двумерное изображение. В 3D-графике спрайты также иногда называют биллбордами (billboard). В Xtreme3D поддерживаются экранные спрайты (отрисовываются на экране поверх проекции) и пространственные (присутствуют в трехмерной сцене в качестве плоскостей). Главное отличие пространственного спрайта от простой плоскости заключается в том, что спрайт всегда повернут в сторону камеры. Это позволяет использовать пространственные спрайты в качестве различных псевдотрехмерных эффектов, такие как взрывы, вспышки, сияния. На их основе можно создать собственную систему частиц.
Экранные спрайты применяются для отображения элементов интерфейса - прицел, иконки, шкала энергии и т.д.
Все спрайты создаются из материалов. При этом поддерживаются их настройки - режимы смешивания, альфа-канал и др.

HUDSpriteCreate

real = HUDSpriteCreate( material as string, width,height,parent as real);
Создает новый экранный спрайт и возвращает его id.
material - имя материала
width,height - ширина и высота спрайта в пикселях
parent - id родителя для спрайта (0 - отсутствие родителя).

SpriteCreate

real = SpriteCreate( material as string, width,height,parent as real);
Создает новый пространственный спрайт и возвращает его id.
material - имя материала
width,height - ширина и высота спрайта в единицах координатной системы
parent - id родителя для спрайта (0 - отсутствие родителя).

SpriteCreateEx

real = SpriteCreateEx( material as string, width,height,left,top,right,bottom,parent as real);
Создает новый пространственный спрайт из фрагмента текстуры материала и возвращает его id. Если у указанного материала нет текстуры, эта функция эквивалентна предыдущей. Функция полезна при создании множества спрайтов сравнительного маленького размера, расположенных на одной большой текстуре - это значительно эффективнее, чем загружать несколько отдельных текстур для каждого спрайта.
material - имя материала
width,height - ширина и высота спрайта в единицах координатной системы
left,top - координаты левого верхнего угла фрагмента (в пикселях, относительно левого верхнего угла текстуры)
right,bottom - координаты правого нижнего угла фрагмента (в пикселях, относительно левого верхнего угла текстуры)
parent - id родителя для спрайта (0 - отсутствие родителя).

SpriteSetSize

real = SpriteSetSize( sprite,width,height as real );
Задает размер спрайта.
sprite - id спрайта
width,height - ширина и высота спрайта.

SpriteScale

real = SpriteScale( sprite,u,v as real );
Изменяет размер спрайта относительно текущего размера.
sprite - id спрайта
u,v - добавляются к текущим значениям ширины и высоты.

SpriteSetRotation

real = SpriteSetRotation( sprite,angle as real );
Задает угол поворота спрайта. Обратите внимание, что этот угол относится только к изображению и никак не влияет на обычный поворот объекта по осям XYZ.
sprite - id спрайта
angle - угол поворота.

SpriteRotate

real = SpriteRotate( sprite,angle as real );
Изменяет поворот спрайта относительно текущего угла. Обратите внимание, что этот угол относится только к изображению и никак не влияет на обычный поворот объекта по осям XYZ.
sprite - id спрайта
angle - добавляется к текущему углу поворота.

SpriteMirror

real = SpriteMirror( sprite,u,v as real );
Отображает спрайт зеркально по горизонтали и/или вертикали. Работает только для пространственных спрайтов.
sprite - id спрайта
u - горизонталь, true или false (1 и 0 соответственно).
v - вертикаль, true или false (1 и 0 соответственно).

SpriteNoZWrite

real = SpriteNoZWrite( sprite,mode as real );
Включает или выключает учет Z-буфера для спрайта. Отключение этого свойства позволяет устранить артефакты, возникающие при перекрытии друг друга несколькими пространственными спрайтами. Это особенно заметно в случае с прозрачными изображениями.
sprite - id спрайта
mode - true или false (1 и 0 соответственно).
